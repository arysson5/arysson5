## ◈ Sobre

Desenvolvedor Full Stack com **4+ anos de experiência** entregando sistemas em produção. Atuo na **BGF Consultoria em Engenharia** desde 2021, construindo soluções que vão de módulos internos com PHP/Laravel até apps mobile com gamificação, e-commerces com IA e chatbots corporativos com LLM.

Meu diferencial é a **visão de produto**: entender o problema de negócio antes de escrever a primeira linha de código. Cada projeto que construo tem um propósito claro e é entregue com qualidade de produção.

```
✔  Backend robusto com PHP, Laravel, Node.js e Python
✔  Frontend moderno com React e React Native (mobile)
✔  IA integrada: Google AI, LLMs, Realidade Aumentada
✔  Deploy em produção: Vercel, Render, CI/CD automatizado
✔  4+ anos de experiência em empresa de engenharia de médio porte
```
