### 🔥 [GARRA](https://github.com/arysson5/GARRA) — App de Hábitos Gamificado

> *O projeto mais ambicioso — app mobile fullstack inspirado no conceito de Grit (garra): perseverança e paixão por objetivos de longo prazo.*

```
Stack:  React Native · Expo Router · Node.js · Express · SQLite · JWT · SecureStore
Plataforma: iOS + Android
```

- 🎮 **Gamificação completa**: XP, níveis, streaks, combo multiplier e moedas de disciplina
- 🌑 **Antihábitos (Shadow)**: evite maus hábitos e seja recompensado por isso
- 👥 **Hábitos em grupo**: crie grupos, convide amigos e acompanhe progresso coletivo
- 🏆 **Rede social**: amigos, ranking e ligas progressivas (Bronze → Prata → Ouro → Diamante)
- 👆 **Swipe rápido**: gesto direita = feito ✅ · gesto esquerda = falhei ❌
- 🔔 **Notificações inteligentes**: resumo matinal (7h) e lembretes noturnos (21h)
- 🍅 **Modo Foco**: timer Pomodoro integrado
- 📊 **Heatmap anual**: grade de consistência estilo GitHub
- 🏅 **Conquistas e onboarding** personalizado por perfil
- 🔐 Autenticação segura com JWT + bcrypt + Expo SecureStore
